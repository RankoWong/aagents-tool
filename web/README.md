# m4b-tool Web 界面使用说明

## 🚀 快速启动

### 方式1：使用启动脚本（推荐）

```bash
cd /Users/wangdafeng/m4b-tool
./web/start.sh
```

### 方式2：手动启动

```bash
/opt/homebrew/Cellar/php@8.2/8.2.30/bin/php -S localhost:8080 -t web/
```

启动后，在浏览器中打开：**http://localhost:8080**

## 📦 合并功能

### 使用步骤：

1. **上传文件**
   - 点击上传区域选择文件
   - 或直接拖拽多个音频文件到上传区域
   - 支持格式：MP3, M4A, M4B, AAC, OGG, FLAC, WAV, WMA

2. **调整顺序**
   - 拖动文件项可以调整合并顺序
   - 点击"删除"按钮可以移除文件

3. **设置选项**（可选）
   - **标题**：有声书的名称
   - **作者/朗读者**：作者或朗读者的名字
   - **专辑**：系列名称
   - **类型**：有声书类型
   - **音频比特率**：选择输出质量（推荐96k或128k）

4. **开始合并**
   - 点击"开始合并"按钮
   - 等待处理完成（可以看到进度和日志）
   - 完成后点击下载链接获取合并后的M4B文件

### 示例：

```
上传文件：
  001_第一章.mp3
  002_第二章.mp3
  003_第三章.mp3

设置：
  标题：哈利波特与魔法石
  作者：J.K. Rowling
  比特率：96k

结果：
  哈利波特与魔法石.m4b (包含3个章节)
```

## ✂️ 分割功能

### 使用步骤：

1. **上传文件**
   - 点击上传区域选择要分割的M4B文件
   - 或直接拖拽文件到上传区域

2. **设置选项**
   - **输出格式**：选择输出格式（MP3/M4B/M4A/AAC）
   - **音频比特率**：选择输出质量
   - **音频通道**：单声道（推荐）或立体声

3. **开始分割**
   - 点击"开始分割"按钮
   - 等待处理完成
   - 完成后可以下载所有分割后的文件

### 示例：

```
上传文件：
  哈利波特与魔法石.m4b (包含10个章节)

设置：
  输出格式：MP3
  比特率：64k
  通道：单声道

结果：
  001-第一章.mp3
  002-第二章.mp3
  003-第三章.mp3
  ...
```

```
└── output/            # 输出文件目录
```

## ⚠️ 注意事项

1. **文件大小限制**
   - PHP默认上传限制为2GB
   - 大文件处理需要等待较长时间
   - 建议单次处理不超过5GB

2. **性能提示**
   - 合并多个文件时，按文件顺序处理
   - 处理时间取决于文件大小和数量
   - 进度每2秒刷新一次

3. **浏览器兼容性**
   - 推荐使用Chrome、Firefox、Safari最新版本
   - 需要支持JavaScript和拖拽API
   - 不支持IE浏览器

4. **清理文件**
   - uploads/目录中的临时文件会在1小时后自动删除
   - output/目录中的输出文件会永久保存
   - 定期清理output目录以节省空间

## 🔧 故障排除

### 问题1：无法上传文件

**解决方案：**
- 检查文件格式是否支持
- 确认uploads/目录有写入权限
- 检查磁盘空间是否充足

### 问题2：处理卡住不动

**解决方案：**
- 打开浏览器开发者工具（F12）查看控制台错误
- 检查PHP进程是否运行
- 查看日志：`/tmp/m4b-web-server.log`

### 问题3：下载的文件无法播放

**解决方案：**
- 确认文件大小正常（不是0字节）
- 尝试使用VLC播放器
- 检查原始文件是否损坏

## 💡 高级用法

### 批量处理

对于多本有声书，可以：

1. 先合并第一本书
2. 下载后刷新页面
3. 继续处理下一本书

### 命令行 + Web界面结合

```bash
# 1. 先用命令行预处理
m4b-tool.sh chapters --adjust-by-silence input.m4b -o adjusted.m4b

# 2. 再用Web界面分割
# 上传 adjusted.m4b 进行分割
```

### 自动化脚本

```bash
#!/bin/bash
# 批量合并多个目录

for dir in audiobooks/*/; do
  cd "$dir"
  m4b-tool.sh merge . --output-file="../$(basename $dir).m4b"
  cd ../..
done
```
